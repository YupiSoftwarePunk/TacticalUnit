# TacticalUnit — Frontend

Фронтенд-часть платформы для игрового сообщества (клана) в Squad. Проект находится в активной стадии разработки.

-----

## Ссылки

  * **Backend Репозиторий:** [github.com/NekichNet/ACCS](https://github.com/NekichNet/ACCS) — API на .NET 10.
  * **Канбан-доска проекта:** [Ссылка на доску](https://github.com/users/NekichNet/projects/1/views/1) — *здесь мы отслеживаем задачи и прогресс.*

-----

## Технологический стек

Мы используем современный и производительный стек для обеспечения быстрого UI и удобной типизации:

  * **Framework:** [Next.js](https://nextjs.org/) (App Router) — для SSR и оптимизации.
  * **Language:** [TypeScript](https://www.typescriptlang.org/) — строгая типизация для минимизации ошибок.
  * **Styling:** [Tailwind CSS 3](https://tailwindcss.com/) — современный подход к верстке через утилитарные классы.
  * **Architecture:** [Feature-Sliced Design (FSD)](https://feature-sliced.design/) — для масштабируемости и чистоты кода.

-----

## Быстрый старт

### Системные требования

  * Node.js 20.x или выше.
  * npm / pnpm.

### Установка и запуск

1.  **Клонируйте репозиторий и перейдите в папку проекта:**
    ```bash
    cd client
    ```
2.  **Установите зависимости:**
    ```bash
    npm install
    ```
3.  **Запустите сервер для разработки:**
    ```bash
    npm run dev
    ```
    Локальный адрес: `http://localhost:3000`.

-----

## Правила разработки

1.  **Стилизация:** Все новые компоненты должны использовать переменные из `@theme` (напр. `text-text-primary`, `bg-bg-primary`), чтобы корректно работала темная и светлая темы.
2.  **Коммиты:** Используйте префиксы `feat:`, `fix:`, `docs:` или `refactor:`.
3.  **Архитектура:** Не импортируйте модули из `features` напрямую в `entities`. Соблюдайте иерархию FSD.
